# Felipe Smith Language (FSL) - Dependencies

# ===== CORE DEPENDENCIES =====
# Este projeto usa APENAS a biblioteca padrão do Python!
# Não há dependências externas necessárias.

# A FSL utiliza:
# - re (regex) - módulo padrão para lexer
# - subprocess - módulo padrão para invocar GCC
# - os - módulo padrão para operações de sistema

# ===== EXTERNAL REQUIREMENTS =====
# GCC (GNU Compiler Collection) deve estar instalado no sistema
# e disponível no PATH para compilar o código C gerado.

# Instalação do GCC:
# - Linux (Debian/Ubuntu): sudo apt-get install gcc
# - macOS: xcode-select --install
# - Windows: MinGW (http://www.mingw.org/) ou MSYS2 (https://www.msys2.org/)
